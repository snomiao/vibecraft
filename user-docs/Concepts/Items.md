# Items

Definition

- Inventory-like UI entries representing resources/tools. Currently UI-only placeholder.

In-Game Behavior

- Hidden by default; inventory section can be toggled via the `inventoryVisible` prop in BottomHUD.
- When visible, displays static mock items in a grid layout.
- No gameplay effects yet.

UI/UX

- The Bottom HUD allows click-through in gaps between sections for world interaction (RTS-style), while HUD panels remain interactive.

Future Hooks

- Action triggers, consumption, effects on units/buildings.
