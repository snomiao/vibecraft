# Support

## Questions and Help

- Open a GitHub discussion for usage questions and general help.
- Open a GitHub issue for reproducible bugs and feature requests.

## Before Opening an Issue

- Confirm the problem still occurs on the latest `main`.
- Include environment details (OS, Node version, Bun version).
- Provide clear reproduction steps and logs/screenshots when useful.

## Security Reports

For security issues, do not open a public issue. See `SECURITY.md`.
